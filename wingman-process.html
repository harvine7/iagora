Homepage


  1. Tombol untuk aktifkan layanan
  2. Tombol untuk menampilkan total pendapatan dan pencairan dana

    • Tombol untuk menampilkan pesanan yang diterima / belum di proses
    • Tombol untuk menampilkan histori pesanan

Total Pendapatan


  1. Tampilan pendapatan per hari ini
  2. Tombol untuk pencairan dana ke rekening yang telah terdaftar saat pembuatan akun.
    Dana akan dicairkan setiap 24 jam setiap :
    Hari : Senin - Sabtu
    Pukul : 8 - 10 pagi
    Disarankan untuk selalu memeriksa rekening untuk memastikan pencairan dana.

Pusat Bantuan WINGMAN (IAGORA Care)



Halaman Chat dengan Pusat Bantuan IAGORA Care
Halaman ini akan muncul jika menekan tombol IAGORA Care pada tombol navigasi bagian bawah di Homepage, ataupun dapat diakses melalui halaman profile pada bagian Pusat Bantuan.
Halaman ini menampilkan fitur chatting dengan Customer Service IAGORA secara real-time dan responsif apabila WINGMAN menemukan kendala. Waktu untuk menunggu balasan dari Customer Service berkisar antara 5-10 menit.



Menerima dan Memproses Pesanan



Halaman pertama :
Akan muncul pop-up ketika pesanan masuk dan dapat melihat detail pesanan berupa alamat customer, jarak tempuh menuju alamat customer, estimasi pendapatan, serta detail daftar belanjaan sebelum menerima / menolak pesanan tersebut.

Halaman kedua :
Menampilkan detail pesanan

  1. Tombol untuk melihat detail pesanan
  2. Tombol untuk menerima / menolak pesanan




Setelah menerima pesanan, maka akan muncul tampilan seperti disamping ini.

Halaman pertama :
Menampilkan data pesanan yang baru diterima agar segera diproses

Halaman kedua :
Setelah menekan data pesanan dari halaman pertama, maka akan muncul detail pesanan customer. tekan tombol
Proses Pesanan untuk mulai memproses.




Setelah menekan tombol Proses Pesanan, maka akan dialihkan ke halaman disamping ini.
  1. Tombol untuk membatalkan pesanan
  2. Tombol untuk menggunakan fitur chat dengan customer
  3. Pada bagian ini terdapat 4 komponen penting, yaitu :
    • Tombol untuk menghapus item pada pesanan
    • Tombol untuk menandai produk yang telah dibeli
    • Field untuk mengubah harga apabila terjadi perubahan harga pada produk
    • Tampilan catatan tambahan customer untuk setiap produk
  4. Tombol untuk mengubah harga ongkir.
  5. Tombol Aktivasi Pengiriman ditekan apabila telah selesai membeli semua produk. Tombol ini muncul jika customer memilih metode pengiriman Wingman Delivery
  6. Jika customer memilih metode Jemput Sendiri, maka tombol yang akan muncul adalah
    Pesanan dapat dijemput. Tekan tombol tersebut apabila telah selesai membeli semua produk.




Halaman ini akan muncul jika menekan tombol chat pada halaman Proses Order
Pada fitur ini, WINGMAN dapat melakukan percakapan dengan customer perihal harga dan kualitas produk dengan sopan dan ramah. WINGMAN juga dapat mengirimkan foto produk pada customer apabila diperlukan. Foto tersebut tidak dapat diambil dari galeri smartphone.
